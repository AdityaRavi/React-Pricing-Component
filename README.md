# React Pricing Component

This is a visual clone of [fontawesome's pricing component](https://fontawesome.com/plans) implemented in React.
The implementation of this component is unrelated to the implementation used by fontawesome.

![React Pricing Component](./public/screenshot.png?raw=true)

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
